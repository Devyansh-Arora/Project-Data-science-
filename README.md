# 🌾 Smart Agriculture Predictor

AI-powered crop recommendation dashboard built with **Streamlit** + **scikit-learn**.

> Dark EcoSync-style UI • 5 ML models • 22 crops • ROI calculator • No Plotly

---

## 🚀 Quick Start (Local)

### Step 1 — Generate Model Files

Open and run **all cells** in `agriproject2_fixed__1_.ipynb`.  
This creates the `models/` folder with all `.pkl` files.

**Alternative** (if you only have raw data, not the trained notebook):
```bash
python generate_models.py
```

### Step 2 — Install Dependencies

```bash
pip install -r requirements.txt
```

### Step 3 — Run the App

```bash
streamlit run app.py
```

Open `http://localhost:8501` in your browser. ✅

## ☁️ Deploy on Streamlit Cloud (GitHub)

### Step 1 — Push to GitHub

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/smart-agri-app.git
git push -u origin main
```

> ⚠️ **models/ folder size**: if `.pkl` files are large, either:
> - Push them to GitHub (fine if < 100 MB total), or
> - Add `models/` to `.gitignore` and use `generate_models.py` + a startup script

### Step 2 — Deploy on Streamlit Cloud

1. Go to **[share.streamlit.io](https://share.streamlit.io)**
2. Sign in with GitHub
3. Click **New App**
4. Select your repo → branch `main` → main file `app.py`
5. Click **Deploy** 🚀

Your app will be live at: `https://your-app-name.streamlit.app`

## 🔗 How PKL Files Connect Notebook → App

```
Jupyter Notebook
    │
    │  trains → rf_cls, lr_cls, dt_cls, knn_cls, svm_cls
    │  fits   → scaler_cls (StandardScaler)
    │  fits   → le_crop (LabelEncoder for crop names)
    │
    ▼
joblib.dump(...) → models/*.pkl
    │
    │  (copy models/ folder to project root)
    │
    ▼
Streamlit app.py
    │
    ├─ joblib.load("models/rf_cls.pkl")      → model
    ├─ joblib.load("models/scaler_cls.pkl")  → scaler
    └─ joblib.load("models/le_crop.pkl")     → label encoder
           │
           ▼
    user sliders → scale → predict → decode → display recommendation
```

## 🌱 Input Ranges

| Feature | Range | Unit |
|---------|-------|------|
| Nitrogen (N) | 0 – 140 | kg/ha |
| Phosphorus (P) | 5 – 145 | kg/ha |
| Potassium (K) | 5 – 205 | kg/ha |
| Temperature | 8 – 44 | °C |
| Humidity | 14 – 100 | % |
| Soil pH | 3.5 – 9.9 | — |
| Rainfall | 20 – 298 | mm (dataset scale) |

> ⚠️ **Rainfall note**: Use dataset-scale values (20–298 mm), NOT annual totals.  
> For Punjab's ~650 mm annual rainfall, enter ~130–160 mm.
